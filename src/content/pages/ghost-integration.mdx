<p><a href="https://ghost.org/?ref=ghost.theor.net">Ghost</a> is an open-source blog &amp; newsletter platform. I maintain my own fork at <a href="https://github.com/theoryzhenkov/repo.ghost?ref=ghost.theor.net">https://github.com/theoryzhenkov/repo.ghost</a>, and host a Ghost instance at <a href="https://ghost.theor.net/">ghost.theor.net</a>. </p><p>At build time, this website fetches Ghost notes, posts and pages, parses metadata, and embeds them on the website. This allows me to have version controlled, remote edited repository for all my posts, notes, pages. </p><p>My version of Ghost extends the base with numerous bug fixes, additional API for ActivityPub and an improved editor. I do not use Ghost built-in observability tools, instead self-hosting a <a href="https://medama.mintlify.app/introduction?ref=ghost.theor.net">Medama</a> instance. </p>
