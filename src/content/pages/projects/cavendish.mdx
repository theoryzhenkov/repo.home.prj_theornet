# Cavendish

[Documentation](https://palisade-research.github.io/cavendish/) · [GitHub](https://github.com/PalisadeResearch/cavendish)

Cavendish is a fullstack service for running software-defined experiments, sharing them through a brew-like registry system, and inspecting results. Built at [Palisade Research](https://palisaderesearch.org).

## Why Cavendish?

Running AI safety evaluations requires reproducible experiment pipelines that can be shared, versioned, and inspected. Cavendish provides the infrastructure — define experiments in code, run them on managed executors, and browse results through a web UI.

## Stack

- **Backend** — Python, FastAPI, SQLAlchemy ORM
- **Frontend** — React SPA with TypeScript and Tailwind CSS
- **Infrastructure** — Containerized deployment with structured logging and observability
