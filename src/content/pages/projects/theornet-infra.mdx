## Hosting

Website is hosted on [Hetzner](https://www.hetzner.com/) servers, machine running on [NixOS](https://nixos.org/). I own domain name [theor.net](theor.net) with [Porkbun](https://porkbun.com/), and manage DNS with [Terraform](https://www.terraform.io/).

The Docker images for applications at `*.theor.net` are deployed to GHCR via a CI pipeline. For more details about deploying a new application to theor.net, see the [deploy](/deploying-to-theor-net-infrastructure) page.
