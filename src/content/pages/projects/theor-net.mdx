# TheoR.net

TheoR.net is a personal web ecosystem — a collection of services and sites under the `theor.net` domain.

## Components

- **home.theor.net** — personal site and project showcase (this site)
- **schedule.theor.net** — public schedule and availability
- **index.theor.net** — link index and directory
- **infra.theor.net** — shared infrastructure, DNS, and deployment configuration
