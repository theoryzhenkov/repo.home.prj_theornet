## Relational structure

```markdown
is:
  project:
links:
  github: https://github.com/theoryzhenkov/...
```

| Key  | Implies | Meaning                                    |
| ---- | ------- | ------------------------------------------ |
| `up` | `down`  | This page is contained in / child of...    |
| `down` | `up`  | This page contains / is parent of...       |
| `is` | `has`   | This page is an instance of / belongs to...  |
| `has` | `is`  | This page is a type/category that includes... |
| `next` | `prev`  | Next page in sequence is ... |
| `prev` | `next`  | Previous page in sequence is ... |
| `ref` | `refi`  | References ... |
| `refi` | `ref` | Is referenced by ... |
